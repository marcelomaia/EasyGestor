Fazer de uma filial a companhia principal

Este processo é utilizado para fazer de uma filial a companhia principal, isto é, a filial matriz, responsável por todas as outras.

  1. No Aplicativo Administrativo abra a pesquisa de parâmetros.

  2. Encontre o parâmetro Companhia principal. Ao dar um duplo clique, você pode realizar a alteração para qualquer outra filial cadastrada. Após ter certeza, clique em Ok para confirmar o processo.

  3. Pronto! Você realizou a alteração da companhia principal com sucesso!
