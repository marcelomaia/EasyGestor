Criando um perfil de usuário

Este processo é utilizado para cadastrar perfis de usuários no Stoq.

  1. No Aplicativo Administrativo, abra a pesquisa de perfis de usuários.

  2. Clique no botão Novo.

  3. Um editor será aberto e nele você deve preencher o Nome do perfil, assim como marcar as caixas de permissão dos aplicativos que poderão ser acessados por este perfil. Ao ter certeza dos dados, clique em Ok para finalizar o processo.

  4. Pronto! Seu perfil de usuário foi criado com sucesso! Você já pode vinculá-lo a um usuário.
