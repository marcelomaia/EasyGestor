Usuários

Cada pessoa que utiliza o Stoq possuirá um usuário cadastrado. Este usuário terá vinculado a si um perfil de usuário que será utilizado para determinar quais aplicativos ele poderá utilizar, além de possívelmente um cargo.

Assim como todas as outras pessoas (físicas ou jurídicas) cadastradas no Stoq, um cliente pode realizar vários papeis ao mesmo tempo, como também ser um fornecedor, um funcionário e etc. Isso facilita a edição de seus dados básicos.
