Alterando o perfil de um usuário

Este processo é utilizado para vincular um usuário a um perfil de usuário. Isto é importante pois, o perfil de um usuário define quais aplicativos ele está apto a utilizar.

  1. Abra o Aplicativo Administrativo e comece a editar o usuário em questão.

  2. Na aba Detalhes do usuário, você pode realizar a alteração do Perfil.
