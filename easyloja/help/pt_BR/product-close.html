Fechando um produto

Este processo é utilizado para fechar um produto do Stoq, isto é, impedir que novas compras/vendas deste sejam feitas. Isso é interessante para produtos que saíram de linha, mas já foram comprados e vendidos no passado. Como o Stoq não consegue remover esses produtos esta é a opção para realizar o mesmo, sem quebrar essas referências.

Note que, para realizar um fechamento de um produto, é necessário que o mesmo não possua estoque físico cadastrado no Stoq. Caso seja necessário, você pode realizar uma saída manual.

  1. Abra o Aplicativo Compras.

  2. Realize uma pesquisa de produtos pelo produto em questão.

  3. Nesta pesquisa, tendo o produto selecionado, clique no botão Editar.

  4. Um editor será aberto com as informações do produto em questão.

  5. Ao lado dos botões Ok e Cancelar, você irá encontrar um botão Fechar. Clique nele!

  6. Um dialogo será aberto para confirmar o fechamento. Tendo certeza de que é isso mesmo que você quer fazer, confirme-o.

  7. Pronto! Seu produto foi fechado com sucesso no Stoq.
