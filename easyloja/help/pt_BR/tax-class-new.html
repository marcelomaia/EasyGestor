Cadastrando uma classe imposto

Este processo é utilizado para cadastrar impostos ICMC/IPI no Stoq.

  1. No Aplicativo Administrativo, abra a pesquisa de classe de impostos.

  2. Clique no botão Novo.

  3. Um editor será aberto. Após escolher um Nome, selecione o tipo do Imposto (ICMS ou IPI). Dependendo do tipo, haverão campos diferentes a serem preenchidos abaixo. Após o preenchimento e verificação dos dados, você pode clicar em Ok para finalizar o processo.

    Note que, dependendo do cadastro da filial, caso esta seja optante pelo Simples Nacional, o campo utilizado para tributação será o CSOSN (Código de Situação da Operação no Simples Nacional). Caso contrário, o campo utilizado será o CST (Código de Situação Tributária).

  4. Pronto! Seu imposto foi criado com sucesso no Stoq! Você já pode vinculá-lo a um produto.
