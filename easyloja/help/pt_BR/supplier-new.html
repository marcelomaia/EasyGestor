Cadastrando um fornecedor

Este processo é utilizado para cadastrar fornecedores no Stoq, isto é, cadastrar uma pessoa (física ou jurídica) que irá fornecer produtos às nossas filiais para suprir seus estoques.

O procedimento abaixo utilizará o Aplicativo Compras para realizar o cadastro, porém, assim como todas as outras pessoas (física ou jurídica) no Stoq, o mesmo pode ser realizado no Aplicativo Administrativo no ícone Fornecedores.

  1. No Aplicativo Compras, abra a pesquisa de fornecedores.

  2. Clique no botão Novo.

  3. Um assistente será aberto:

    1. No primeiro passo, você deve indicar se está adicionando Uma nova pessoa física ou Uma nova pessoa jurídica. Após isso, clique em Avançar.

      Tenha certeza de inserir o número de telefone correto do fornecedor, pois desse jeito, caso ele já possua um cadastro (como um cliente por exemplo) os dados gerais (como nome, telefone, cpf/cnpj, endereços) serão compartilhados entre ambos, facilitando assim a manutenção dos dados.

    2. No segundo passo você deve informar os dados desse fornecedor, como o seu Nome, Endereço e etc. As informações estão dívididas em abas e podem ser diferentes de acordo com o tipo de fornecedor. No caso de se tratar de uma pessoa física, uma aba existira contendo Dados de pessoa física, enquanto no caso de pessoa jurídica essa aba se chamará Dados de pessoa jurídica, além de abas extras referente à um fornecedor. Após o preenchimento dos campos obrigatórios e correção dos erros de validação o botão Terminar será habilitado e você pode finalizar o processo.

  4. Pronto! Seu fornecedor foi criado com sucesso no Stoq!
