Registrando uma chamada para cliente

Este processo é utilizado para cadastrar chamadas para clientes no Stoq, isto é, informar ao Stoq que em certa data foi realizada uma chamada para um certo cliente para tratar sobre um assunto em específico.

  1. No Aplicativo Vendas, abra a pesquisa de chamadas para clientes.

  2. Clique no botão Novo.

  3. Um editor será aberto e nele você deverá preencher dados como a Pessoa que foi realizada a ligação, o Atendente responsável, assim como uma Data e Descrição. Ao ter certeza dos dados, clique em Ok para finalizar o processo.

  4. Pronto! Seu chamada para cliente foi registrada com sucesso no Stoq.
