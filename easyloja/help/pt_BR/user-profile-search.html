Pesquisando por um perfil de usuário

Este processo é utilizado para encontrar perfis de usuários cadastrados no Stoq.

  1. Abra o Aplicativo Administrativo.

  2. Selecione o menu Pesquisar ▸ Perfis de usuários.

  3. Uma pesquisa será aberta e nela você pode encontrar o perfil em questão. Utilize filtros para facilitar a pesquisa.
