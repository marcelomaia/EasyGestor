Pesquisando por um usuários

Este processo é utilizado para encontrar usuários cadastrados no Stoq.

  1. Abra o Aplicativo Administrativo.

  2. Selecione o menu Pesquisar ▸ Usuários.

    Alternativamente, você pode realizar o mesmo na barra de ferramentas em Buscar ▸ Usuários.

  3. Uma pesquisa será aberta e nela você pode encontrar o usuário em questão. Utilize filtros para facilitar a pesquisa.
