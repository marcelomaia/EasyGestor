Adicionando um fornecedor para um produto

Este processo é utilizado para adicionar um fornecedor à um produto, isto é, indicar ao Stoq que um dado produto é fornecido por um dado fornecedor.

Isso é importante pois, para realizarmos um pedido de compra a um fornecedor, para adicionarmos um produto à esta compra, é necessário que o Stoq saiba que ele é fornecido por pelo mesmo.

  1. Abra o Aplicativo Compras e realize uma pesquisa de produtos pelo produto em questão.

  2. Nesta pesquisa, tendo o produto selecionado, clique no botão Editar.

  3. Um editor será aberto com as informações do produto em questão.

  4. Navegue até a aba Fornecedores. Nela, selecione o fornecedor que você deseja adicionar e clique em Adicionar.

  5. Um dialogo será aberto e nele você poderá preencher alguns dados sobre o fornecimento dese produto por este fornecedor, como o Custo do fornecedor, o Mínimo para compras e etc. Após o preenchimento dos campos obrigatórios e correção dos erros de validação o botão Ok será habilitado, e você poderá confirmar o diálogo.

    Esses dados são importantes pois serão utilizados para preenchimento automático e validação na hora de realizar uma compra. Ex. O Stoq não deixará um usuário realizar uma compra de 5un de um produto X caso esteja indicado 10un no campo Mínimo para compras. Neste caso, ele precisaria realizar o pedido de no mínimo 10un.

  6. Após ter adicionado todos os fornecedores pretendidos e ter certeza de que tudo está Ok, clique em Ok para confirmar o processo.

  7. Pronto! Seu produto foi vinculado ao fornecedor com sucesso!
