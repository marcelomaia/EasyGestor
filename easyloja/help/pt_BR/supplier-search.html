Pesquisando por um fornecedor

Este processo é utilizado para encontrar fornecedores cadastrados no Stoq.

O procedimento abaixo utilizará o Aplicativo Compras para realizar o cadastro, porém, assim como todas as outras pessoas (física ou jurídica) no Stoq, o mesmo pode ser realizado no Aplicativo Administrativo no ícone Fornecedores.

  1. Abra o Aplicativo Compras.

  2. Selecione o menu Pesquisar ▸ Fornecedores.

    Alternativamente, você pode realizar o mesmo na barra de ferramentas em Buscar ▸ Fornecedores.

  3. Uma pesquisa será aberta e nela você pode encontrar o fornecedor em questão. Utilize filtros para facilitar a pesquisa.
