Pesquisando por uma filial

Este processo é utilizado para encontrar filiais cadastradas no Stoq.

  1. Abra o Aplicativo Administrativo.

  2. Selecione o menu Pesquisar ▸ Filiais.

  3. Uma pesquisa será aberta e nela você pode encontrar a filial em questão. Utilize filtros para facilitar a pesquisa.
