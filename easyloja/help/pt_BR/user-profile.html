Perfis de usuários

Um perfil de usuário é um conjunto de permissões de acesso aos aplicativos do Stoq.

Com essa diferenciação, podemos ter por exemplo um perfil chamado "Administrador", onde os usuários desse perfil podem utilizar todos os aplicativos, enquanto um perfil "Vendedor" apenas poderia utilizar o Aplicativo Ponto de vendas e Aplicativo Caixa.
