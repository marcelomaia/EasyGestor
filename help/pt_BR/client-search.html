Pesquisando por um cliente

Este processo é utilizado para encontrar clientes cadastrados no Stoq.

O procedimento abaixo utilizará o Aplicativo Vendas para realizar o cadastro, porém, assim como todas as outras pessoas (física ou jurídica) no Stoq, o mesmo pode ser realizado no Aplicativo Administrativo no ícone Clientes.

  1. Abra o Aplicativo Vendas.

  2. Selecione o menu Pesquisar ▸ Clientes.

    Alternativamente, você pode realizar o mesmo na barra de ferramentas em Buscar ▸ Clientes.

  3. Uma pesquisa será aberta e nela você pode encontrar o cliente em questão. Utilize filtros para facilitar a pesquisa.
