Realizando a contagem de um inventário

Este processo é utilizado para informar ao Stoq a contagem de um inventário aberto. Isso pode ser facilitado se você utilizar uma impressão de listagem de produtos previamente preenchida por um funcionário e apenas transpassá-la.

Assim como todas as operações de inventário, enquanto um inventário estiver aberto, não é possível realizar operações que alterem o estoque de produtos.

  1. No Aplicativo Inventário realize uma pesquisa de inventários pelo inventário em questão e selecione-o.

  2. Tendo o inventário selecionado, selecione o menu Inventário ▸ Contar.

    Alternativamente, você pode realizar o mesmo na barra de ferramentas no botão Contar.

  3. No editor que será aberto, você deve preencher as atuais quantidade de cada produto do inventário. Após isso, clique em Ok para confirmar o processo.

  4. Pronto! Você informou a contagem ao Stoq com sucesso! Agora você já pode realizar um ajuste do mesmo.
