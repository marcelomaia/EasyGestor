Fechando um serviço

Este processo é utilizado para fechar um serviço do Stoq, isto é, impedir que sejam realizadas novas vendas deste. Isso é interessante para serviços que não são mais prestados pela empresa, mas já foram no passado. Como o Stoq não consegue remover esses serviços esta é a opção para realizar o mesmo, sem quebrar essas referências.

Note que o serviço de entrega apontado nos parâmetros não pode ser fechado. Neste caso, aponte outro serviço como serviço de entrega.

  1. Abra o Aplicativo Compras.

  2. Realize uma pesquisa de serviços pelo serviço em questão.

  3. Nesta pesquisa, tendo o serviço selecionado, clique no botão Editar.

  4. Um editor será aberto com as informações do serviço em questão.

  5. Ao lado dos botões Ok e Cancelar, você irá encontrar um botão Fechar. Clique nele!

  6. Um dialogo será aberto para confirmar o fechamento. Tendo certeza de que é isso mesmo que você quer fazer, confirme-o.

  7. Pronto! Seu serviço foi fechado com sucesso no Stoq.
