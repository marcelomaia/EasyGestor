Cadastrando uma transportadora

Este processo é utilizado para cadastrar transportadoras no Stoq.

O procedimento abaixo utilizará o Aplicativo Compras para realizar o cadastro, porém, assim como todas as outras pessoas (física ou jurídica) no Stoq, o mesmo pode ser realizado no Aplicativo Administrativo no ícone Transportadoras.

  1. No Aplicativo Compras, abra a pesquisa de transportadoras.

  2. Clique no botão Novo.

  3. Um assistente será aberto:

    1. No primeiro passo, você deve indicar se está adicionando Uma nova pessoa física ou Uma nova pessoa jurídica. Após isso, clique em Avançar.

      Tenha certeza de inserir o número de telefone correto da transportadora, pois desse jeito, caso ele já possua um cadastro (como um fornecedor por exemplo) os dados gerais (como nome, telefone, cpf/cnpj, endereços) serão compartilhados entre ambos, facilitando assim a manutenção dos dados.

    2. No segundo passo você deve informar os dados dessa transportadora, como o seu Nome, Endereço e etc. As informações estão dívididas em abas e podem ser diferentes de acordo com o tipo de transportadora. No caso de se tratar de uma pessoa física, uma aba existira contendo Dados de pessoa física, enquanto no caso de pessoa jurídica essa aba se chamará Dados de pessoa jurídica, além de abas extras referente à uma transportadora. Após o preenchimento dos campos obrigatórios e correção dos erros de validação o botão Terminar será habilitado e você pode finalizar o processo.

  4. Pronto! Sua transportadora foi criada com sucesso no Stoq!
