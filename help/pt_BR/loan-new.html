Realizando um empréstimo

Utilizamos este processo para realizar um empréstimo de produtos a um dado cliente.

Note que, enquanto o produto está emprestado a um cliente, seu estoque será automáticamente diminuído.

  1. Abra o Aplicativo Estoque ou Aplicativo Vendas.

  2. Selecione o menu Arquivo ▸ Novo ▸ Empréstimo.

    Alternativamente, você pode realizar o mesmo barra de ferramentas em Novo ▸ Empréstimo.

  3. Um assistente será aberto:

    1. No primeiro passo você deverá informar o Cliente que está retirando os produtos, assim como um limite para que ele os devolva em Expira.

      Note que, caso esse seja um novo cliente, você pode clicar no botão Adicionar (botão + ao lado) para adicionar um novo cliente.

    2. No segundo passo você deve selecionar cada Item que será emprestado, prestando atenção na Quantidade e Preço de venda. Este será utilizado caso o cliente decida ficar com o produto e comprá-lo. Após ter adicionado os itens, clique em Terminar para finalizar o processo.

  4. Um dialogo aparecerá perguntando se você deseja imprimir um recibo. Este, caso imprimido, deve ser assinado pelo cliente para firmar um contrato informal de que este está ciente do empréstimo e deve devolver ou comprar os itens até a data de expiração.

    Note que, caso o parâmetro Impressão de notas promissórias em empréstimos seja verdadeiro, uma nota promissória será impressa junto ao recibo do empréstimo.

  5. Pronto! O seu empréstimo foi realizado com sucesso!
