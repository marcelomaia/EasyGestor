Configurando e controlando maquinetas de cartão

Ao realizar vendas com a forma de pagamento cartão, é possível configurar os custos que serão gerados para cada transação realizada.

Sumário

  • Configurando
  • Usando

Configurando

Para configurar os custos que cada maquineta de cartão tem, faça:

  1. Acesse o aplicativo Administrativo

  2. Selecione a opção Formas de pagamento

  3. Selecione a opção Cartão e clique em Editar

  4. Selecione a aba Dispositivos de cartão e edite uma das máquinetas já cadastradas ou crie uma nova.

  5. Ao editar uma maquineta, é possível adicionar os diversos custos para cada tipo de operação. Você precisará informar: Fornecedor de crédito (Visa, Amex, etc.), Tipo de cartão (Crédito, Débito, etc..), e numero de parcelas (se for um tipo parcelado). Para cada situaçõa possível, você indicará o número de dias para o recebimento do pagamento e taxas e tarífas que possam existir.

Usando

Ao realizar uma venda com a forma de pagamento cartão, será possível indicar qual é a maquineta que está sendo utilizada. Ao registrar os pagamentos, o Stoq irá verificar a configuração das condições indicadas (fornecedor, tipo de cartão e número de parcelas), e irá registrar também os custos que foram gerados, além de calcular corretamente a data de vencimento das parcelas.

Caso as condições indicadas não tenham sido configuradas, o vencimento dos pagamentos serão calculados com base na data atual, e não será criado nenhum custo derivado desses pagamentos
