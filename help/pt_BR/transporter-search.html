Pesquisando por uma transportadora

Este processo é utilizado para encontrar transportadoras cadastrados no Stoq.

O procedimento abaixo utilizará o Aplicativo Compras para realizar o cadastro, porém, assim como todas as outras pessoas (física ou jurídica) no Stoq, o mesmo pode ser realizado no Aplicativo Administrativo no ícone Transportadoras.

  1. Abra o Aplicativo Compras.

  2. Selecione o menu Pesquisar ▸ Transportadoras.

  3. Uma pesquisa será aberta e nela você pode encontrar a transportadora em questão. Utilize filtros para facilitar a pesquisa.
