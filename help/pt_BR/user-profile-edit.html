Editando um perfil de usuário

Este processo é utilizado para editar informações de perfis de usuários cadastrados no Stoq.

  1. No Aplicativo Administrativo, abra a pesquisa de perfis de usuários, procure pelo perfil em questão e selecione-o.

  2. Tendo o perfil selecionado, clique em Editar.

  3. Um editor será aberto e você poderá editar as informações desse perfil, como as permissões de acesso a aplicativos. Após ter certeza de tudo, clique em Ok para confirmar o processo.

    Note que a qualquer momento caso você se arrependa da edição, você pode clicar em Cancelar e assim descartar todas as mudanças realizadas no momento.

  4. Pronto! Seu perfil de usuário foi editado com sucesso no Stoq!
