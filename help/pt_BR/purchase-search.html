Pesquisando por uma compra

Este processo é utilizado para encontrar pedidos de compra realizados no Stoq.

  1. Abra o Aplicativo Compras

  2. Você pode encontrar os pedidos na lista de resultados do aplicativo. Utilize filtros para facilitar a pesquisa.
