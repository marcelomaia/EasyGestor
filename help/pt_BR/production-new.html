Criando uma ordem de produção

Este processo é utilizado para criar uma nova ordem de produção de produtos de produção no Stoq.

Após criada a produção, é necessário iniciá-la e posteriormente, informar os itens produzidos.

  1. Abra o Aplicativo Produção.

  2. Selecione Arquivo ▸ Novo ▸ Ordem de produção.

    Alternativamente, você pode criar um produto na barra de ferramentas em Novo ▸ Ordem de produção.

  3. Um assistente será aberto:

    1. No primeiro passo, você preencherá as informações básicas da produção como, qual a Filial que irá produzir, a Data prevista para início, uma Descrição e etc. Após o preenchimento dos campos obrigatórios e correção dos erros de validação o botão Avançar será habilitado e você pode ir para o próximo passo.

    2. No segundo passo, você deve selecionar os serviços que serão utilizados para realizar esta produção. Após ter certeza de todos, clique em Avançar.

    3. No terceiro passo, você deve selecionar os produtos que serão produzidos nessa produção. Após ter certeza de todos, clique em Avançar.

      Note que os produtos que aparecem aqui não são produtos simples, mas sim os produtos de produção registrados anteriormente.

    4. No terceiro passo você deverá verificar os itens que serão produzidos. Após isso, clique em Terminar para finalizar o processo.

      Note que, se a caixa Iniciar a produção mesmo se houver falta de componentes estiver marcada, o Stoq te deixará iniciciar a produção, porém, não te deixará finalizá-la até que todos os componentes possam ser consumidos.

  4. Pronto! Seu ordem de produção foi registrada com sucesso no Stoq.
