Transações

Transações financeiras podem representar uma conta paga, uma conta recebida, uma operação fiscal de um caixa e até mesmo uma transação de banco que foi importada, entre outras.

Elas são organizadas dentro de contas para maior facilidade de uso e administração.

O stoq utiliza o método das partidas dobradas para a organização de transações financeiras. Isso significa que cada transação possui entrada em uma conta e saída em outra. Você pode ver mais informações aqui: Método das partidas dobradas.
