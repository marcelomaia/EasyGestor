Pesquisando por uma classe de imposto

Este processo é utilizado para encontrar classes de impostos cadastradas no Stoq.

  1. Abra o Aplicativo Administrativo.

  2. Selecione o menu Pesquisar ▸ Classes de impostos.

  3. Uma pesquisa será aberta e nela você pode encontrar a classe de imposto em questão. Utilize filtros para facilitar a pesquisa.
