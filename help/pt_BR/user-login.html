Acessando o Stoq

Este processo é utilizado para acessar o Stoq tendo assim acesso ao Lançador de aplicativos e podendo utilizar os diversos aplicativos oferecidos.

Tenha certeza de ter criado o usuário que irá acessar o sistema antes de realizar os procedimentos abaixo.

  1. Ao acessar o Stoq, você irá se deparar com o controle de acesso.

  2. Preencha o seu Usuário e Senha e então pressione Ok.

    A caixa Lembre-se de mim só deve ser marcada no caso de o utilizador ser único no computador que está sendo utilizado. Caso contrário, um outro usuário poderia realizar um acesso maldoso e prejudicar este com a senha salva.

  3. Caso o seu usuário e senha estejam corretos você cairá no Lançador de aplicativos. Ao finalizar o seu acesso, lembre-se de realizar um logout através do menu Arquivo ▸ Encerrar a sessão.
