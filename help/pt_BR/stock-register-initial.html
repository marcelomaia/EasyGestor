Registro de estoque inicial

Este processo é utilizado para registrar um estoque inicial para um certo produto, isto é, registrar uma quantidade inicial que já possuímos desse produto e que não foi recebido de nenhuma compra conhecida pelo Stoq.

Esse processo só pode ser realizado para produtos que nunca foram comprados antes.

  1. Abra o Aplicativo Estoque.

  2. Selecione Arquivo ▸ Registrar estoque inicial.

  3. Um diálogo será aberto, listando todos os novos produtos do Stoq. Na coluna Estoque inicial, preencha a quantidade inicial para cada produto.

  4. Após o preenchimento, clique em Ok para confirmar o processo.

  5. Pronto! O registro de estoque inicial foi realizado com sucesso.

Nesse processo também é possível informar o custo médio inicial para os produtos. Esse custo médio será atualizado nas futuras movimentações dos produtos e será utilizado para calcular o valor total do estoque.
