Alterando a senha de um usuário

Este processo é utilizado para alterar a senha de um usuário, isto é, alterar a palavra passe que ele utiliza junto ao seu nome de usuário para acessar o Stoq.

Existem 2 processos para realizar a alteração. 1 deles por parte do usuário e outro por parte do administrador (ou qualquer um que tenha acesso ao Aplicativo Administrativo).

Pelo usuário

  1. Abra o Lançador de aplicativos.

  2. Selecione o menu Arquivo ▸ Alterar senha.

    Lembre-se que a senha será alterada para o usuário atualmente logado no sistema.

Pelo administrador

  1. Abra o Aplicativo Administrativo e comece a editar o usuário em questão.

  2. Na aba Detalhes do usuário, você pode realizar a alteração no botão Alterar senha.
