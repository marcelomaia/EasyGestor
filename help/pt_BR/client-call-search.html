Pesquisando por uma chamada para um cliente

Este processo é utilizado para encontrar chamadas para clientes registradas no Stoq.

  1. Abra o Aplicativo Vendas.

  2. Selecione o menu Pesquisar ▸ Chamadas para clientes.

  3. Uma pesquisa será aberta e nela você pode encontrar a chamada em questão. Utilize filtros para facilitar a pesquisa.
