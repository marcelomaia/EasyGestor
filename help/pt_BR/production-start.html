Iniciando uma produção

Este processo é utilizado para informar ao Stoq sobre o início de uma ordem de produção.

  1. Abra o Aplicativo Produção, realize uma pesquisa de ordem de produções pela ordem em questão e selecione-a.

  2. Tendo ela selecionada, na lista de resultados selecione o menu Produção ▸ Iniciar produção.

    Alternativamente, você pode realizar o mesmo clicando com o botão direito no item da lista de resultados e selecionando Iniciar produção ou através da barra de ferramentas clicando no botão Iniciar.

  3. Um editor será aberto onde você poderá confirmar e editar a listagem de componentes necessários para a produção. Após ter certeza de todos os dados, clique em Iniciar produção para confirmar o processo.

  4. Pronto! Sua ordem de produção foi marcada como inicada com sucesso! Não se esqueça de finalizá-la, informando a quantidade produzida/perdida de produtos.
