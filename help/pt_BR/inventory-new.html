Abrindo um inventário

Este processo é utilizado para abrir um novo inventário no Stoq. Isso é necessário para que posteriormente possamos realizar uma contagem e um ajuste do estoque de produtos.

Assim como todas as operações de inventário, enquanto um inventário estiver aberto, não é possível realizar operações que alterem o estoque de produtos.

  1. Abra o Aplicativo Inventário.

  2. Selecione o menu Arquivo ▸ Novo ▸ Inventário.

    Alternativamente, você pode realizar o mesmo na barra de ferramentas em Novo ▸ Inventário.

  3. Clique no botão Novo.

  4. No editor que será aberto, você poderá selecionar a Filial em que o inventário está sendo aberto e os produtos que fazem parte deste. Após ter certeza de tudo, clique em Abrir.

    Note que produtos sem categoria não aparecem por padrão. Caso você deseja incluí-los, tenha certeza de marcar a caixa Incluir produtos sem categoria.

  5. Pronto! Seu inventário foi aberto com sucesso! Caso deseje, você pode imprimir uma listagem de produtos desse inventário e entregá-lo a um funcionário para facilitar a contagem.
