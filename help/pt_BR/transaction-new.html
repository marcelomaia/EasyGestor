Criando uma transação

Este processo é utilizado para criar uma nova transação avulsa em uma conta.

  1. No Aplicativo Financeiro dê um duplo clique na conta em que a transação será criada.

  2. Uma aba com o mesmo nome da conta será aberta e selecionada.

  3. Selecione o menu Arquivo ▸ Novo ▸ Transação.

    Alternativamente, você pode realizar o mesmo na barra de ferramentas em Novo ▸ Transação.

  4. Clique no botão Novo.

  5. No editor que será aberto, você deve preencher informações como Valor, Descrição e etc. Caso o valor dessa transação seja negativa, selecione o combo Diminuir, caso contrário, deixe selecionado Adicionar. Após ter certeza dos dados, clique em Ok para confirmar o processo.

  6. Pronto! Sua transação foi criada com sucesso!
