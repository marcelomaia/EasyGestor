Novidades da versão 1.8

Data do lançamento: 2014-03-17

Óticas

  • Novo plugin especializado para uso em óticas.

  • Dados de produtos como Lente de Contato, Lente Oftálmica e Armações.

  • Cadastro de médicos, incluindo CRM.

  • Cadastro da receita do cliente ao realizar uma nova venda, atrelando esta a uma ordem de serviço.

  • Cadastro de anamnese para um melhor acompanhamento dos clientes.

Vendas

  • Novo parâmetro para não aceitar devoluções de clientes sem CPF.

  • Para orçamentos de vendas, é possível exigir a senha de um superior ao aplicar um desconto acima do permitido.

  • Agora é possível inserir comentários em vendas a qualquer momento.

  • Nova opção para criar vendas que exijam ordens de serviços.

  • Nova opção para reservar produtos do estoque em uma venda ainda não confirmada.

  • Novo widget para facilitar o calculo de descontos no wizard de vendas.

  • Possibilidade de verificar o estoque de outras filiais durante uma venda.

  • Agora é permitido fazer uma devolução de venda em outras filiais.

  • Melhoras significativas no wizard de vendas para bancos de dados com muitos clientes.

  • Todos os descontos autorizados são registrados e podem ser visualizado no aplicativo administrativo.

Produtos

  • Suporte para produtos com lotes.

  • Nova opção de criar produtos sem controle de estoque.

  • Melhoras na pesquisa de produtos durante a venda, incluindo busca de palavras com acentuação e fora de ordem.

  • Novos campos Marca e Família no cadastro de produtos.

  • Mostrando a quantidade de produtos vendidos de um empréstimo no histórico de produtos.

Pagamentos

  • Nova opção de pagamento: Crédito, que pode ser gerado quando um cliente faz uma devolução.

  • Após uma devolução e geração de crédito, o cliente pode levar uma “carta de crédito”, caso não queira fazer uma nova compra na hora.

  • Opção de configurar um dispositivo padrão para cada operadora de cartão.

  • Agora também é possível configurar o número de parcelas máximas para cada operadora individualmente.

  • Novo relatório de movimento diário de caixa, listando todos os pagamentos recebidos no dia.

  • Novo campo para número de autorização do cartão, com a possibilidade de tornar o campo obrigatório.

Serviços

  • O aplicativo ‘Manutenção’ foi renomeado para ‘Serviços’.

  • Nova funcionalidade para envio e recebimento de ordem de serviço, útil para empresas onde a ordem de serviço é executada em outra filial.

  • Visão KanBan (experimental), permitindo uma fácil troca de estado das ordens de serviço.

Transferências

  • Transferências agora são feitas em dois passos: Primeiro o envio é feito pela filial de origem, e depois o recebimento é acusado na filial de destino.

  • Aviso de transferências pendentes no aplicativo estoque, além de uma nova busca de transferencias.

Inventário

  • Novo processo de contagem para inventário, usando leitor de código de barras. Além de diversas melhorias para aumentar a performance dos diálogos.

  • Possibilidade de abrir inventário de produtos, baseado em outros critérios.

  • Agora é possível fechar um inventário sem ter que ajustar todos os itens.

Diversos

  • Correções de alguns problemas na geração de txt da NF-e.

  • Suporte para ICMS CSSOSN 900.

  • Cadastro de pessoas agora pede pelo documento (CPF/CNPJ), e não o telefone.

  • Novo parâmetro para logout automático após período de inatividade.

  • É possível editar pedidos de compras já confirmados para corrigir alguns detalhes.

  • Diversas melhoras no diálogo de detalhes de clientes.

  • Empréstimos: Agora é possível fechar mais de um empréstimo ao mesmo tempo.
