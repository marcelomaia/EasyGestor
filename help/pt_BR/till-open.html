Abrindo o caixa

Este processo é utilizado para abrir o caixa, isto é, iniciar as operações fiscais do dia.

Utilizaremos o Aplicativo Caixa neste guia, porém, o mesmo pode ser feito no Aplicativo Ponto de vendas por comodidade.

Note que esta operação só pode ser realizada uma vez por dia. Portanto, evite fechar o caixa até o final do período fiscal.

  1. Abra o Aplicativo Caixa.

  2. Selecione Arquivo ▸ Abrir caixa.

  3. No editor que será aberto, coloque o valor em dinheiro que o caixa está sendo suprido. Após isso, clique em Ok.

  4. Pronto! O caixa foi aberto com sucesso no Stoq.
